Для того, чтобы заработать 1 миллион рублей, 
необходимо правильно ответить на 15 вопросов из 
области знаний "Экономика". Каждый вопрос имеет 4 
варианта ответа, из которых только один является
верным. Каждый вопрос имеет конкретную стоимость...
Все суммы являются заменяемыми, то есть после ответа
на следующий вопрос не суммируются с суммой за 
ответ на предыдущий. 

Подсказки
На сегодняшний день игроку предлагаются 3 подсказки,
каждую из которых можно использовать только 
один раз:
1) «Помощь зала» — Каждый зритель в студии голосует 
за правильный, на его взгляд, ответ, и игроку
 предоставляется статистика голосования.
2) «50:50» — компьютер убирает два неправильных
 ответа, предоставляя игроку выбор из оставшихся
 вариантов ответа.
3) «Звонок другу» — игрок может посоветоваться либо
 с другом по телефону, либо со зрителем в студии.
